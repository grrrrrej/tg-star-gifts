## ✨ Ключевые возможности

| Возможность | Описание |
|------------|----------|
| 🖥️ **Полная имитация ПК** | Использует параметры устройства HP Laptop и ОС Windows 11 x64, что снижает риск блокировки |
| 🎨 **Интеллектуальный интерфейс** | Современные динамические таблицы с цветовой индикацией статусов (библиотека Rich) |
| 🛠️ **Кастомные подарки** | Добавляйте любые подарки через команду `/set [ID] [Название]` |
| 👤 **Анонимность** | Поддержка отправки как анонимно, так и с текстовым сообщением |
| 📦 **Zero-Setup** | Автоматическая установка зависимостей при первом запуске |
| 💾 **Умная сессия** | Сохранение сессии в `session.txt` — повторный вход не требуется |
| 🛡️ **Контроль баланса** | Автоматическая проверка баланса звезд перед отправкой |
